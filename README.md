# cusco-urban-transit
Cusco Urban Transit es un sistema modular desarrollado en C++ que modela, gestiona y analiza operaciones del transporte urbano en la ciudad del Cusco. Incluye componentes para la administración de rutas, unidades vehiculares, conductores, horarios, cálculo de tiempos de viaje y análisis mediante algoritmos avanzados (Dijkstra, BFS, heurísticas.
